🚀 Overview
The IndiSpeech Application is an AI-driven multilingual speech translation tool that converts spoken words into text, translates them into the target language, and generates audio output of the translation. It is designed to bridge communication gaps by enabling seamless speech-to-speech translation across multiple languages.

With a user-friendly interface, the application allows users to input audio via voice recording or file upload, process the speech using Natural Language Processing (NLP) techniques, and get instant text and audio translations in their desired language.

🔥 Key Features
✅ Speech-to-Text Conversion – Converts spoken words into written text using Google Speech-to-Text API.
✅ Language Detection & Translation – Auto-detects the source language and translates text using Google Translate API.
✅ Text-to-Speech Synthesis – Converts translated text into natural-sounding speech using Google Text-to-Speech (gTTS).
✅ Multiple Input Options – Supports audio file uploads and real-time voice recording.
✅ Multi-Language Support – Works with a wide range of languages for both input and output.
✅ Fast & Lightweight – Built on Flask, ensuring a responsive and scalable experience.
✅ Audio Storage – Saves processed audio files in the system for future reference.

🛠️ Technologies Used
Flask – Lightweight web framework for backend processing.
Google Speech-to-Text API – For converting speech into text.
Google Translate API – For real-time translation.
gTTS (Google Text-to-Speech) – For generating speech output.
HTML, CSS, JavaScript – For an intuitive and interactive UI.



🏗️ Installation & Setup
1️⃣ Clone the Repository
bash
git clone https://github.com/your-username/IndiSpeech.git
cd IndiSpeech
2️⃣ Install Dependencies
Ensure you have Python 3.8+ installed. Then, install the required packages:

bash
pip install -r requirements.txt
3️⃣ Run the Application
bash
python app.py
The application will be available at: http://127.0.0.1:5000/

📢 How to Use
Upload an audio file or record your voice directly.
Select the target language for translation.
Click "Translate" to process the speech.
View the translated text and play the translated speech output.
